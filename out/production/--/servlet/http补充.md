# MIME类型说明

> MIME 是 HTTP 协议中数据类型。
>
> MIME 的英文全称是 " Multipurpose I nternet Mail Extensions" 多功能 I nternet 邮件扩充服务。 MIME 类型的格式是“大类型 / 小 类型” ， 并与某一种文件的扩展名相对应。
